1, 启动tftp工具，将Current Directory目录设为需要上传到路由器/网关设备上的文件路径2，网关/路由器启动telnet, 通过xshell/secureCRT登录设备，用户名:CMCCAdmin,密码：aDm8H%MdA或者wE%AnmFq3，路由器可以选择将文件传输到versionbackup目录下， 网关可以选择上传到userConfig目录下，执行命令tftp -l filename -g ip4, 将该文件解压，执行tar -xvzf filename, 路由器进入platformInter/route，网关进入platforminter/gateway5, 修改device_description文件中的terminalId和topology字段为自己设备的SN+MAC, 并将该文件上传至platformInter/route或者platformInter/gateway下6, cd platformInter/route, 执行命令：./xjsdk
